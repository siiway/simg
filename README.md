# simg

Image Download and Hosting

> 图片存储在 `img` 分支

~~> 修改中: 使用 GitHub API 上传文件~~

**Usage / 使用**: python3 dl.py `count` `last` `r18` `auto_upload`

- `const`: 起始数字 (int)
- `last`: 结束数字 (int)
- `r18`: *是否获取 r18 图片* **(别开!!!)** (`true` / `false`)

> 是文本, 不是 `True` / `False`

- `auto_upload`: (int: 1 / 0)
